# Kannur Gardens – Luxury Event Venue

Official website for Kannur Gardens, a premier luxury event venue in Kannur, Kerala.

## Features
- **Luxury Spaces**: AC Banquet Hall, Jasmine Lawn, and Lotus Pavilion.
- **Premium Amenities**: In-house catering, decor, and ample parking.
- **Mobile Optimized**: Cinematic mobile experience with instant WhatsApp inquiry.
- **SEO Ready**: Optimized meta tags and semantic HTML.

## Tech Stack
- React + Vite
- TypeScript
- Tailwind CSS
- Framer Motion (Animations)
- Lucide React (Icons)

## Development
```sh
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment
This project is configured for deployment on **Vercel**.
1. Push to GitHub.
2. Import in Vercel.
3. Done.
